  1. Моделирование поведения агентов в трёхмерном пространстве

    Авторы: С.О. Брезицкий, А.А. Полупанов

    Описание: Статья посвящена моделированию поведения агентов в трёхмерной среде с применением современных инструментов. Разработана экспериментальная сцена эвакуации из здания, выявлено влияние паники на время эвакуации.

    Источник: Прикладная математика: современные проблемы математики, информатики и моделирования / Материалы VII Всероссийской научно-практической конференции молодых ученых. — Краснодар, КубГУ — 2025, с. 311-315.

  2. Подсистема защиты графических данных с помощью объектов со скрытым идентификационным слоем

    Авторы: С.О. Брезицкий, А.А. Полупанов

    Описание: В статье рассмотрены основные методы внедрения скрытого идентификационного слоя в графические объекты. Разработана подсистема для скрытого внедрения идентификационных данных в графические объекты без ухудшения их визуальных характеристик и обеспечения возможности их последующей верификации.

    Источник: Прикладная математика: современные проблемы математики, информатики и моделирования / Материалы VI Всероссийской научно-практической конференции молодых ученых. — Краснодар, КубГУ — 2024, Том I, с. 333-337.

  3. Доклад по теме "KAN"

    Авторы: С.О. Брезицкий

    Описание: Доклад посвящён сравнению классической архитектуры нейронных сетей (MLP) и новой архитектуры — KAN (Kolmogorov–Arnold Networks).

  4. Доклад по теме "Практическое применение агентов"

    Авторы: С.О. Брезицкий

    Описание: Доклад исследует применение агентного моделирования для оценки эффективности мобильного приложения по отслеживанию контактов во время пандемии COVID-19.

  5. Доклад по теме "Анализ агентных платформ"

    Авторы: С.О. Брезицкий

    Описание: Представляет обзор платформы SPADE для разработки мультиагентных систем на Python. Описываются её основные особенности и приводятся примеры реальных проектов.

  8. Обзор эволюционных методов оптимизации на основе роевого интеллекта
